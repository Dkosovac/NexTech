# Agregati Mašine – Monitoring Dashboard

A static web-based monitoring dashboard for generators (power aggregates) and industrial machines.

Live demo:  
https://agregatimasine.netlify.app/

---

## Overview

This project is a **front-end monitoring dashboard** designed to visualize the operational status of generators and industrial machines.

The focus is on:
- clear system status visualization,
- operational metrics overview,
- warning and critical state indicators,
- control-center / terminal-inspired UI.

The application is currently **static** and intended as:
- a UI prototype for SCADA / IoT systems,
- a foundation for future backend integration,
- a demonstration or educational project.

---

## Technologies

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Netlify (Hosting)

No backend, database, or server-side logic is included at this stage.

---

## Features

- Centralized system dashboard
- Generator status indicators (Operational / Warning / Critical)
- System metrics visualization (simulated)
- Terminal-style, engineering-oriented UI
- Lightweight and fast (no frameworks)
